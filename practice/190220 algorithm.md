### 1-1. 부록

- pop 의 주의사항 - 비어있는 list에서 실행하면 오류.

```python
# push의 방법인데, 여기서 중요한 점은 함수(push) 안에서 사용한 변수(top, stack)가 함수 밖에서도 저장된다는 점이다.(global 사용으로 변수를 데려옴, list 같은 reference 타입은 global을 안써도 된다고 하긴 함. 하지만 그냥 다 해도 됨.)

size = 100
stack = [0]*100
top=-1
def push(item):
    global top, stack
    if top >= size -1:
        return
    else:
        top += 1
        stack[top]=item

```

위의 코드를 보면, 

함수(def)의 리턴값을 꼭 result로 가야한다는 고정관념이 깨지게 된다. 

return을 통해 더 진행가능한 함수의 상태를 끝냄으로서, 가지치기가 가능한 형태로도 사용가능하다. 글로벌 변수를 최종적으로 출력하기를 원하거나 글로벌 변수가 특정 조건을 만족 할 때에 함수를 끝내고 싶다면, return 다음에 변수가 아니라 단순히 return 만 입력함으로써 함수를 종료하고 print를 통해 글로벌 변수를 출력하는 것이 가능하다.
