## 1. 2차원 배열

### 1.1 간단한 예시

a=[[0,1,2,3],[4,5,6,7]] 이런 식으로 표현이 가능하다



- 그럼 3차원은??
  - a = [ [ [1,2,3], [4,5,6], [7,8,9] ] ]

  ```python
  a[0][0][0]=1
  a[0][0][1]=2
  a[0][0][2]=3
  a[0][1][0]=4
  이런 식으로 적용됨
  ```
